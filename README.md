# gs-read-me-later
PWA to save links to read later

## Frontend

- VueJS
- List of links saved
- Buttons for create, update, delete items

## Backend

- GoLang
- CRUD for links saved
- share endpoint for receive/update link
- Link detection: URL, Title, keywords
